Chatoboto

Um bot de Discord inteligente, alimentado pela API do Claude da Anthropic. Conversas naturais, memória por canal, e respostas que fazem sentido.

⚡ discord.js v14 ✓ Node.js 20+ 🤖 Claude Sonnet 🇧🇷 PT-BR

✨ Funcionalidades

🧠
IA Contextual
Memória de conversa por canal com histórico rolante de 20 mensagens.
⚡
Resposta Rápida
Indicador "digitando..." enquanto o Claude processa para feedback instantâneo.
💬
Menção Direta
Responde somente quando mencionado — sem spam no servidor.
📄
Mensagens Longas
Divide automaticamente respostas maiores que 2000 caracteres.

🛠️ Stack

🟢 Node.js Runtime JavaScript para o servidor v20+
🔵 discord.js Wrapper oficial para a API do Discord v14
🤖 Anthropic SDK Integração com modelos Claude latest
🔐 dotenv Gerenciamento de variáveis de ambiente v16

🚀 Como Rodar

1
Clone e instale as dependências
Clone o repositório e instale os pacotes necessários.
bash 
git clone https://github.com/seu-usuario/chatoboto
cd chatoboto
npm install
2
Configure as variáveis de ambiente
Crie um arquivo .env na raiz do projeto:
.env 
DISCORD_TOKEN=seu_token_aqui
ANTHROPIC_API_KEY=sk-ant-...
⚠️ Nunca suba o .env para o Git. Adicione ao .gitignore.
3
Ative o Message Content Intent
No Discord Developer Portal, vá em Bot → Privileged Gateway Intents e ative MESSAGE CONTENT INTENT.
4
Inicie o bot
bash 
node src/index.js
✅ Você verá Bot online como Chatoboto#0000 no terminal.

💡 Como Usar

Mencione o bot em qualquer canal do servidor onde ele foi adicionado:

discord 
# O bot responde quando mencionado
@Chatoboto Olá! Quem foi Alan Turing?

# O bot mantém contexto da conversa
@Chatoboto E qual foi sua contribuição para a computação?

# Para resetar o histórico do canal, reinicie o bot
